introdução ao css

existem 3 formas de vincular ocss aos elementos

css externo:

podemos vincular qualquer arquivo html a um arquivo externo de css.

essa e a forma correta de criar sites, principalmenre inde ha mais de uma pagina . pois o arquivo css fica separado do html e vinculamos o mesmo css a todas as paginas html.

css dentro do html

podemos adicionar estilos css diretamente no arquivo html ,usando tag style.

css inline :

e possivel adicionar estilos css diretamente em cada tag, usando o atributo style
